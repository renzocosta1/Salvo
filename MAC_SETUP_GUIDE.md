# 🍎 Mac Setup Guide for Salvo

Quick setup guide for running Salvo on your Mac after pulling from GitHub.

---

## 📋 Prerequisites

- [ ] Node.js installed (v18+)
- [ ] Git installed
- [ ] Cursor IDE installed
- [ ] Expo Go app on your iPhone

---

## 🚀 Setup Steps

### 1. Clone Repository

```bash
cd ~/Documents  # or wherever you want the project
git clone https://github.com/renzocosta1/Salvo.git
cd Salvo
```

### 2. Copy Environment Variables

**CRITICAL**: You need to copy your `.env` file from Windows to Mac.

**Option A - Manual Copy** (Recommended):
1. On Windows, open `c:\Coding Projects\Salvo\.env`
2. Copy the entire contents
3. On Mac, create `.env` in the project root:
   ```bash
   nano .env  # or use Cursor to create it
   ```
4. Paste the contents
5. Save and exit (Ctrl+X, Y, Enter for nano)

**Option B - Via Cloud** (if you have access):
- Copy `.env` to iCloud/Dropbox/Google Drive from Windows
- Download to Mac and place in project root

**Required API Keys** (already in your .env):
- `GOOGLE_API_KEY` - For Gemini AI verification
- `PERPLEXITY_API_KEY` - For Taskmaster research
- `EXPO_PUBLIC_GOOGLE_CIVIC_API_KEY` - For Google Civic API
- `EXPO_PUBLIC_SUPABASE_URL` - Your Supabase instance
- `EXPO_PUBLIC_SUPABASE_ANON_KEY` - Supabase public key
- `EXPO_PUBLIC_MAPBOX_ACCESS_TOKEN` - For maps
- `EXPO_PUBLIC_VAPID_PUBLIC_KEY` - For PWA push notifications
- `VAPID_PRIVATE_KEY` - For PWA push notifications

### 3. Install Dependencies

```bash
npm install
```

This will install all packages from `package.json`.

### 4. Start Expo Dev Server

```bash
npx expo start
```

**What you'll see**:
- QR code in terminal
- Metro bundler running on port 8081 (or next available)
- Options: press `i` for iOS simulator, `a` for Android, `w` for web

### 5. Connect Your iPhone

**Option A - Scan QR Code**:
1. Open Expo Go app on iPhone
2. Tap "Scan QR Code"
3. Scan the QR code in terminal
4. App will load

**Option B - Same Network**:
1. Make sure Mac and iPhone on same WiFi
2. In Expo Go, app should appear under "Recently opened"
3. Tap to open

### 6. Verify Everything Works

Test these features:
- [ ] Login with your `renzorodriguez2001@gmail.com` account
- [ ] Navigate to War Room - should show 2024 deadlines (COMPLETED)
- [ ] Navigate to Ballot - should show Anne Arundel District 32 ballot
- [ ] Navigate to Admin → Endorsements - should have access
- [ ] Test endorsing candidates - should save successfully

## 🔧 Troubleshooting

### "Cannot find module" errors
```bash
# Clear cache and reinstall
rm -rf node_modules
npm install
npx expo start --clear
```

### Port conflicts
```bash
# Kill existing Expo process
pkill -f "expo"
# Or use different port
npx expo start --port 8082
```

### iPhone not connecting
- Make sure both devices on same WiFi
- Check firewall isn't blocking port 8081
- Try scanning QR code again
- Restart Expo Go app

### Missing .env errors
- Make sure `.env` file exists in project root
- Check all required keys are present
- No extra spaces or quotes around keys
